# Tool Descriptions for Soccer Knowledge Graph Agent

## Cortex Analyst Tool

**Tool Name:** `soccer_semantic_model`

**Description:** This semantic model provides structured SQL-based access to soccer knowledge graph data including players, clubs, matches, contracts, and performance statistics. Use this tool for queries about player profiles, club information, match results, transfer histories, contract details, team rosters, performance metrics, and any questions that can be answered with SQL aggregations or joins. Query this tool when users ask about player statistics, club histories, match outcomes, roster compositions, or any data retrieval that doesn't require complex network analysis.

---

## Custom Tools (Graph Analytics)

### 1. Shortest Path Tool

**Tool Name:** `shortest_path_tool`

**Description:** This graph analytics function finds the shortest connection path between two players or clubs in the soccer network. Use this tool to discover how players are connected through shared clubs, teammates, or match participations. Query this tool when users ask about player connections, degrees of separation, or relationship paths between entities.

---

### 2. Centrality Analysis Tool

**Tool Name:** `centrality_tool`

**Description:** This graph analytics function identifies the most influential players or clubs in the network using centrality metrics (betweenness, degree, eigenvector, closeness). Use this tool to rank entities by their network importance, influence, or connectivity. Query this tool when users ask about most influential players, key connectors, network hubs, or player/club importance rankings.

---

### 3. Community Detection Tool

**Tool Name:** `community_detection_tool`

**Description:** This graph analytics function detects natural groupings and communities within player or club networks based on connection patterns. Use this tool to identify player clusters, team ecosystems, or transfer network communities. Query this tool when users ask about player groups, club affiliations, network clusters, or community structures in soccer networks.

---

### 4. Transfer Network Analysis Tool

**Tool Name:** `transfer_analysis_tool`

**Description:** This graph analytics function analyzes player transfer patterns and movement history for specific clubs or players within a date range. Use this tool to trace transfer chains, analyze club recruitment patterns, or examine player career trajectories. Query this tool when users ask about transfer histories, player movements between clubs, recruitment patterns, or transfer network dynamics.

---

### 5. Temporal Analysis Tool

**Tool Name:** `temporal_analysis_tool`

**Description:** This graph analytics function examines how player and club networks evolve over time, showing trends in transfers, contract activity, and network structure changes. Use this tool to identify temporal patterns, market trends, or historical network evolution. Query this tool when users ask about network changes over time, transfer trends, historical patterns, or time-series analysis of soccer networks.
